# Дракоша — Звёздный питомец

Образовательная веб-игра для детей 6–11 лет. Фиджитал-механика: ребёнок получает задание → выполняет в реальном мире → система валидирует → питомец растёт.

## Быстрый старт

### Требования
- Node.js 18+
- npm 9+

### Установка и запуск

```bash
# Клонировать репозиторий
git clone <url>
cd uchi_ru

# Установить зависимости
npm install
cd client && npm install && cd ..
cd server && npm install && cd ..

# Настроить API ключ (опционально — без него работает в оффлайн-режиме)
cp server/.env.example server/.env
# Добавить ANTHROPIC_API_KEY в server/.env

# Запустить проект
npm run dev
```

Фронтенд: http://localhost:5173  
API: http://localhost:3001

### Только фронтенд (без сервера)
```bash
cd client && npm run dev
```
Игра работает и без API-ключа: валидация речи и фото переключается на оффлайн-режим.

## Игровой цикл

1. **Создание персонажа** — ребёнок вводит имя и знакомится с Дракошей
2. **Главный экран** — питомец с тремя шкалами потребностей (Знания / Творчество / Энергия)
3. **Выбор задания** — нажать на шкалу → получить задание
4. **Выполнение задания:**
   - **Чтение** — прочитать текст вслух (Web Speech API)
   - **Творчество** — нарисовать/сфотографировать (Claude Vision)
   - **Активность** — физическое упражнение по таймеру
5. **Награда** — XP, конфетти, рост питомца
6. **Родительская панель** — статистика, фото работ, прогресс

## Технологии

| Слой | Технологии |
|------|-----------|
| Фронтенд | React 19, TypeScript, Vite, Tailwind CSS, Framer Motion |
| Бэкенд | Node.js, Express, TypeScript |
| База данных | SQLite (better-sqlite3) |
| Распознавание речи | Web Speech API (встроен в браузер) |

## Контент для детей

- **18 заданий** 3 типов (чтение, фото, активность)
- **5 стадий роста** питомца (смена изображения по уровням)
- **Система XP** с уровнями (экспоненциальный рост)
- **Родительская панель** с историей и фото работ

## Оффлайн-режим

Игра полностью работает без сервера:
- Состояние хранится в localStorage
- Валидация речи — алгоритм сравнения слов
- Валидация фото — автопринятие (педагогически безопасно)
